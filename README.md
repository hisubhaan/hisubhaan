## How HTML, CSS, and JavaScript Are Used

### 1) HTML (Structure)
HTML defines the structure and content of the site, including:
- Navigation and page sections (Home, About, Skills, Projects, Contact)
- Text content and headings
- Images, icons, and links
- Forms and input fields for contact

### 2) CSS (Design + Layout)
CSS controls the visual style and layout, including:
- Light/Dark theme colors using CSS variables
- Typography, spacing, and grid layouts
- Buttons, cards, and responsive design
- Smooth animations and hover effects

### 3) JavaScript (Interaction)
JavaScript adds dynamic behavior, such as:
- Theme toggle (Light/Dark) with local storage
- Mobile navigation open/close
- Scroll-based reveal animations
- Smooth scrolling for anchor links
- Contact form submission status
